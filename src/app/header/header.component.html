SymplVote

A simple polling app created with Angular 5 and Ruby on Rails Api

User Stories Process
SympleVote User Stories
  • As a user I should be able to view current polls.
  • As a user I should be able to create and delete my polls.
  • As a user I should be able to Up the vote on a specific poll.
  • As a user my polls and votes should persist in the database.
